> **행정과 현장을 이해하는 전문 파트너. 아이들의 꿈과 미소를 지키는 제주 돌담입니다.**

제주 돌담은 제주도 현장체험학습 안전관리(주간 안전관리, 야간 생활지도)를 전문으로 하는 기업입니다. 체계적인 시스템과 검증된 인력을 통해 가장 안전한 체험학습 환경을 조성합니다.

## ✨ 주요 기능 및 특징
- **Professionalism**: 100% 국가공인 자격 보유자 배치
- **Real-time Care**: 24시간 안심 케어 시스템 (Phase 01-03 운영)
- **Tactical Process**: 4단계 전술 타임라인을 통한 즉각적인 대응 시스템
- **Structural Safety**: 데이터 기반의 정밀 안전계획 수립
- **Responsive Design**: 모든 기기(Mobile, Tablet, Desktop)에 최적화된 웹사이트

## 🛠 Tech Stack
- **Frontend**: React 19, TypeScript, Tailwind CSS
- **Icons**: Lucide React

## 📁 Project Structure
- `/components`: UI 컴포넌트 (Hero, Navbar, Overview, Procedure 등)
- `App.tsx`: 메인 어플리케이션 로직
- `index.html`: 메타 태그 및 글로벌 스타일 설정
